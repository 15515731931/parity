Parity FIX Gateway
==================

Parity FIX Gateway adds a Financial Information Exchange (FIX) interface to
Parity Trading System.


Features
--------

Parity FIX Gateway specifies the following protocols:

  - [**FIX**](doc/FIX.md): an alternative order entry protocol
